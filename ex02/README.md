### 追加機能
- 四則演算:＋以外の四則演算をする。*, /の文字盤表記を×, ÷にする。
- オールクリア:Cのキーをクリックすると, entryをすべて削除
- redo,undo:「↶」キーを押すと, キャッシュされたデータがentryに復元する。キャッシュは=キーとCキー,「↷」キーを押した場合に行われる。「↷」は「↶」の逆の動作をする。
